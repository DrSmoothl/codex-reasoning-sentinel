# Reasoning Guard Plugin

This directory contains the bundled Codex plugin implementation for Codex Reasoning Sentinel.

For project documentation, installation instructions, configuration, and limitations, see:

- [English README](../../README.md)
- [简体中文 README](../../README.zh-CN.md)

Install from the repository root:

```bash
codex plugin marketplace add "$PWD"
codex plugin add reasoning-guard@codex-reasoning-sentinel
```
